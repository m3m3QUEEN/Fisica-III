## 📚 Contenido del Módulo

El repositorio se enfoca en la exploración computacional del **Movimiento Oscilatorio**:

1.  **Movimiento Armónico Simple (MAS):** Modelado de posición, velocidad y aceleración.
2.  **Sistemas Dinámicos:** Simulaciones de sistemas masa-resorte y diversos tipos de péndulos.
3.  **Análisis Energético:** Visualización interactiva de la conservación y transferencia de energía.
4.  **Oscilaciones Reales:** Modelado de amortiguamiento y fenómenos de resonancia.
