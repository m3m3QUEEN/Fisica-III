# Cuaderno Interactivo: Movimiento Oscilatorio

Este repositorio contiene un **cuaderno interactivo sobre Movimiento Oscilatorio**, diseñado como material de apoyo para el estudio de conceptos fundamentales de Física 3.

El repositorio incluye todos los recursos necesarios para visualizar correctamente el cuaderno virtual:

- 📘 **Cuaderno virtual** con la explicación teórica del movimiento oscilatorio.
- 🖼️ **Carpeta de imágenes** utilizadas dentro del cuaderno.
- 🧪 **Simulaciones interactivas** que permiten explorar el comportamiento del sistema oscilatorio.
- 📂 **Archivos estructurales** necesarios para el funcionamiento del cuaderno.

Este cuaderno interactivo tiene como objetivo facilitar la comprensión del **movimiento oscilatorio**, integrando teoría, visualizaciones y simulaciones para reforzar el aprendizaje de manera dinámica.
